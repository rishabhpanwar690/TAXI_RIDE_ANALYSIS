# 🚕 Taxi Ride Analytics Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Taxi Ride Analytics Dashboard** built using **Power BI**.  
The dashboard analyzes ride booking data to understand customer behavior, revenue trends, cancellations, vehicle performance, ride distance patterns, and customer satisfaction.

The project focuses on converting raw ride booking data into meaningful business insights through data cleaning, transformation, visualization, and analytics.

---

## 📂 Dataset Information

Dataset: `rideBookings.csv`

The dataset contains **150,000 ride booking records** with information about:

- Booking details
- Customer information
- Vehicle categories
- Pickup & drop locations
- Ride status
- Cancellation details
- Payment methods
- Revenue
- Distance travelled
- Driver and customer ratings

---

## 📑 Dataset Features

| Column | Description |
|---|---|
| Date | Ride booking date |
| Time | Booking time |
| Booking ID | Unique ride identifier |
| Booking Status | Completed / Cancelled / Incomplete status |
| Customer ID | Unique customer identifier |
| Vehicle Type | Type of vehicle booked |
| Pickup Location | Starting location |
| Drop Location | Destination |
| Avg VTAT | Vehicle Turn Around Time |
| Avg CTAT | Customer Turn Around Time |
| Booking Value | Ride revenue |
| Ride Distance | Distance covered |
| Driver Ratings | Driver feedback score |
| Customer Rating | Customer feedback score |
| Payment Method | Payment mode used |
